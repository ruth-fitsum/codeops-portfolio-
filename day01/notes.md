# Five Terminal commands
**mkdir** - creates a folder
**cd** - moves to a different folder
**ls** - lists all the files and folder in the folder that I am on 
**ls -la** - list everythingincluding hidden files 
**echo** - when used with either > or >> it writes a text on a file specified next
