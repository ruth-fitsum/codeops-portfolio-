# Five Terminal commands
-**mkdir** - creates a folder \n
-**cd** - moves to a different folder \n
-**ls** - lists all the files and folder in the folder that I am on \n
-**ls -la** - list everythingincluding hidden files \n
-**echo** - when used with either > or >> it writes a text on a file specified next \n

### Three Git Commands
-**git log** - to view the commit history \n
-**git init** - to initate git on the folder that I am working on \n
-**git clone** - to copy or download all the files and codes that are from a github repo \n
